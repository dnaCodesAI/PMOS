# PRD — PRFAQ Format

Produce a complete PRD with these sections:
1. Problem Statement — user perspective, no solution mentioned
2. Press Release — written as if it has shipped
3. Internal FAQ — 5 questions engineering will ask
4. Customer FAQ — 5 questions users will ask
5. User Stories — Given/When/Then format
6. Acceptance Criteria — specific, testable, falsifiable
7. Success Metrics — primary metric + counter-metric
8. Out of Scope — explicit list
